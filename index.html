Code Studio helpt Sint Lucas studenten met vragen over het digitale medium.

Code Studio is een intiatief waar studenten naar toe kunnen komen met vragen over digitale media. Vragen zoals:

  • Hoe maak ik een portfolio?
  • Hoe maak ik een installatie die reageert op geluid?
  • Hoe visualiseer ik Twitter-data?

Code Studio is voor studenten uit alle jaren en alle richtingen; van eerstejaars grafische vormgevers die help willen met het maken van een boekje tot masterstudenten vrije kunsten die een interactieve installatie willen bouwen. We helpen studenten in het hele proces, van de conceptuele fase, waar we kunnen helpen met bepalen wat haalbaar is, tot de uitwerking en de presentatie.

Code Studio wordt wekelijks op woensdagvoormiddag (10-13u) georganiseerd op Sint Lucas Antwerpen in het Atelierhuis (eerste verdieping).
